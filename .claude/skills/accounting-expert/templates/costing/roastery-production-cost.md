# TÍNH GIÁ THÀNH SẢN XUẤT CÀ PHÊ RANG (ROASTERY COSTING)

Mẫu này giúp xác định giá vốn chính xác cho 1kg cà phê thành phẩm, tính đến hao hụt và chi phí sản xuất.

## 1. Công thức tính giá vốn (COGS per kg)

`Giá vốn = (Giá mua hạt xanh / % Tỷ lệ thu hồi) + Chi phí bao bì + Chi phí sản xuất trực tiếp`

## 2. Bảng tính chi tiết (Mẫu)

| Hạng mục | Giá trị | Đơn vị | Ghi chú |
|----------|---------|--------|---------|
| **A. Nguyên liệu đầu vào (Hạt xanh)** | | | |
| 1. Giá mua hạt xanh (Green bean) | 150,000 | VNĐ/kg | Đã bao gồm vận chuyển |
| 2. Tỷ lệ hao hụt khi rang (Loss) | 15% - 18% | % | Tùy độ rang (Light/Dark) |
| 3. Tỷ lệ thu hồi (Recovery = 1 - Loss)| 82% | % | |
| **Giá hạt sau rang (1 / 3)** | **182,927** | VNĐ/kg | |
| **B. Bao bì & Vật tư** | | | |
| 4. Túi cà phê (Van 1 chiều) | 5,000 | VNĐ/túi | Cho túi 1kg/500g |
| 5. Nhãn mác, tem niêm phong | 1,000 | VNĐ/bộ | |
| **C. Chi phí sản xuất (Direct Labor/Utility)** | | | |
| 6. Điện, gas vận hành máy rang | 2,000 | VNĐ/kg | Ước tính theo công suất xưởng |
| 7. Nhân công rang xay (Trực tiếp) | 3,000 | VNĐ/kg | |
| **TỔNG GIÁ THÀNH (A + B + C)** | **193,927** | **VNĐ/kg** | **Giá vốn tại kho** |

## 3. Quản trị biên lợi nhuận (Margin)

- **Giá vốn:** 193,927 VNĐ
- **Giá bán mục tiêu (B2B sỉ):** 280,000 VNĐ
- **Lợi nhuận gộp:** 86,073 VNĐ
- **% Biên lợi nhuận gộp:** 30.7%
