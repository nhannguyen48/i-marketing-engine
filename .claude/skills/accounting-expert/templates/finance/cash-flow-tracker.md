# MẪU THEO DÕI DÒNG TIỀN (CASH FLOW TRACKER) - EXCEL READY

Mẫu này giúp theo dõi dòng tiền mặt và tiền gửi ngân hàng hàng ngày, giúp CEO nắm bắt khả năng thanh khoản.

## Cấu trúc bảng (Sheet: Daily Cash Flow)

| Ngày | Nội dung | Phân loại | Thu (+) | Chi (-) | Tồn cuối ngày | Ghi chú |
|------|----------|-----------|---------|---------|---------------|---------|
| 01/01| Thu tiền khách hàng A | Sales | 5,000,000 | | | Hợp đồng sỉ |
| 01/01| Chi tiền điện xưởng | Operating | | 2,000,000 | | |
| 02/01| Nhập hàng hạt Arabica | COGS | | 10,000,000 | | |
| ... | | | | | | |

## Danh mục phân loại (Category mapping)

### 1. Dòng tiền từ hoạt động kinh doanh (Operating)
- **Thu:** Doanh thu bán lẻ, doanh thu sỉ, thu hồi công nợ.
- **Chi:** Nhập hàng, lương nhân viên, tiền thuê mặt bằng, điện nước, marketing.

### 2. Dòng tiền từ hoạt động đầu tư (Investing)
- **Chi:** Mua máy rang, máy pha, bàn ghế showroom.
- **Thu:** Thanh lý tài sản cũ.

### 3. Dòng tiền từ hoạt động tài chính (Financing)
- **Thu:** Nhận vốn góp, vay ngân hàng.
- **Chi:** Trả nợ gốc, trả lãi vay, chia cổ tức.
